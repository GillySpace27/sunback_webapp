#
# Keeps sunpy/sunkit-image#324 (sort-and-group rhef kernel, ~10× at our bin
# density) + #325 (analytic find_pixel_radii fast path on non-rotated HPC maps,
# ~7×) — together ~2.1 s → ~150 ms for a 4096² RHEF on the Render box. Those
# PRs are NOT in a tagged PyPI release yet, so plain PyPI would forfeit 2–10×.
#
# We no longer `pip install git+…@branch`: pip's partial ("promisor remote")
# clone of the fork flaked ~half of Render's no-cache builds
# (`fatal: early EOF / index-pack failed`). Installing a committed wheel removes
# the build-time git clone entirely → deterministic builds, same fast kernels.
#

# stripped (6.2 MB → 81 KB; runtime never touches sunkit_image/data/test/).

#   # then strip non-.py files under sunkit_image/data/test/ from the .whl and
